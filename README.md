# AcSO — Experiment Code and Data

Reproducibility package for the paper *"An Ontology-Driven, Explainable Approach
to Supervisor Recommendation and Risk Detection in Graduate Thesis Supervision."*

Everything reported in the paper is computed at run time from a **seeded
synthetic cohort** (seed = 42) and the AcSO ontology. No result is hard-coded.

## Install & run

```bash
pip install -r requirements.txt
./run_all.sh                 # or run the four scripts in src/ individually
```

Run individually (from `src/`):

```bash
python evaluate.py                  # E1 recommendation + E2 tracking; writes data/
python graph_embedding_baseline.py  # graph-embedding baseline metrics
python sensitivity.py               # sensitivity + noise robustness; writes fig_sensitivity.png
python make_figures.py              # writes fig_rec_results / fig_ndcg_sweep / fig_tracking
```

## What each script computes

**evaluate.py** — loads `ontology/acso.ttl`, builds the topic taxonomy and the
Wu–Palmer similarity, generates 60 supervisors and 300 students, and defines an
independent gold standard (taxonomic path distance + availability + quality,
deliberately a different measure from Wu–Palmer to avoid circularity). It then
evaluates four recommenders (AcSO full, semantic-only ablation, TF-IDF/cosine,
Jaccard) with Precision/Recall/F1@5, MAP, nDCG@{1,3,5,10}, and a paired Wilcoxon
test; and the tracking rules (multi-condition vs naive) with the full confusion
matrix. Results go to `data/results.json`; the cohort is exported to CSV.

**graph_embedding_baseline.py** — a modern semantic baseline: Laplacian-Eigenmaps
node embeddings of the topic graph (broaderTopic + relatedTopic), cosine ranking.

**sensitivity.py** — (a) nDCG@5 as the quality weight α varies; (b) tracking
metrics as the contact-gap threshold varies; (c) noise robustness: nDCG@5 when a
fraction of observed topic labels is randomly corrupted (mean ± std over 5 seeds).

**make_figures.py** — redraws the recommendation, nDCG-sweep, and tracking figures
from the computed numbers.

## Expected headline numbers (seed = 42)

| Method | P@5 | nDCG@5 |
|---|---|---|
| TF-IDF / cosine | 0.651 | 0.657 |
| Jaccard | 0.623 | 0.646 |
| Graph-embedding | 0.654 | 0.643 |
| AcSO (similarity-only) | 0.655 | 0.663 |
| **AcSO (full)** | **0.968** | **0.974** |

Tracking (AcSO multi-condition vs naive): accuracy 0.737 vs 0.400; false
positives 44 vs 172; base rate of late students 0.287. Wilcoxon p < 0.001.

## Notes

- The dataset is **synthetic** (a proof-of-concept cohort), not real student
  records. To use real data, replace the generation blocks in `evaluate.py`
  (sections 2 and 6) with your own loader, keeping the same fields.
- Topic labels in the CSVs are human-readable; internally topics are ontology
  IRIs under `http://purl.org/acso#`.
- The ontology path can be overridden with the `ACSO_ONTOLOGY` environment
  variable. Re-running reproduces identical numbers (minor ±0.001 rounding may
  appear in the noise-robustness means depending on platform RNG).
